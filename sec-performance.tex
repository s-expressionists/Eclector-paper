\subsection{Benchmarks}
\label{sec:benchmarks}

We measured the time it took the native readers of different \cl{}
implementations as well as extrinsic \eclector{} in those same
implementation to read the contents of \cl{} files taken from
real-word open source libraries.  Code similar to the following was
used to perform the benchmarking:

\begin{verbatim}
(let ((content (alexandria:read-file-into-string filename)))
  #+sbcl (sb-ext:gc :full t)
  #+ccl  (gc)
  (the-cost-of-nothing:benchmark
   (with-input-from-string (stream content)
     (loop for form = (read stream nil stream)
           until (eq form stream)))))
\end{verbatim}
